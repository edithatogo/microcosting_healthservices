## How to use the docs

1. Start with the [2025 version](/versions/2025/) for the current release
   contract.
2. Check [calculator coverage](/governance/calculator-coverage/) to see which
   source families are implemented versus still pending.
3. Read the [governance hub](/governance/) for the rules around validation,
   release, and delivery.
4. Use the legacy migration page only as provenance for the old docs entry
   points.

## Site signals

The site is built with Starlight, uses Pagefind search, keeps versioned docs
available, and is intended to stay conservative about parity claims. The docs
tell you what the tool actually does today, not what future tracks may enable.
