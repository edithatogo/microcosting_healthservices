# Code of Conduct

This project is committed to respectful and constructive collaboration.

## Expected behavior

- Be professional and direct in technical discussions.
- Keep comments specific and actionable.
- Respect data-use boundaries and licensing constraints.
- Do not share private or restricted IHACPA materials.

## Unacceptable behavior

- Personal attacks or harassment
- Threats, discrimination, or abuse
- Misrepresenting source data or release evidence

## Enforcement

For conduct concerns, flag issues with project maintainers using GitHub security or standard issue channels.
Moderation actions will be proportional to impact and documented where needed.
