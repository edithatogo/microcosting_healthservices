# Specification: Cost Bucket Analytics Tutorials

## Overview
Add tutorials showing how cost bucket information can support costing studies, benchmarking, and cost-versus-NWAU analysis. Tutorials should use synthetic data and public aggregate NHCDC tables, not confidential patient-level submissions.

## Functional Requirements
- Demonstrate cost bucket distribution analysis by stream and product type.
- Demonstrate observed cost versus NWAU-funded revenue by cost bucket category.
- Demonstrate public aggregate NHCDC table interpretation and limitations.
- Demonstrate local cost bucket mapping overlays with synthetic examples.
- Link to AHPCS process concepts and the cost bucket registry.

## Non-Functional Requirements
- Tutorials must avoid implying public aggregate data is patient-level costing data.
- Synthetic examples must be clearly labelled.
- Analytical caveats from cost bucket review material must be visible.

## Acceptance Criteria
- Docs include at least two cost bucket tutorials.
- Examples are runnable or checked with synthetic/public-safe fixtures.
- Tutorials distinguish analysis support from official IHACPA costing compliance.

## Source Evidence
- Costing overview: https://www.ihacpa.gov.au/what-we-do/costing
- NHCDC public sector: https://www.ihacpa.gov.au/health-care/costing/national-hospital-cost-data-collection/national-hospital-cost-data-collection-public-sector
- Cost Bucket Review 2025: https://www.ihacpa.gov.au/sites/default/files/2025-11/cost_bucket_review_2025.pdf
