# Health Standards

## Near-Term Standards

- ICD-10-AM.
- ACHI.
- ACS.
- AR-DRG.
- HL7 v2, especially ADT-style patient-administration feeds.
- FHIR R4 as the practical API baseline.
- IHE PAM, PDQ, PIX, PIXm, PDQm, and PMIR for patient identity and
  demographics workflows.

## Watch-List Standards

- ICD-11.
- FHIR R5.
- openEHR for a future clinical repository or archetype/template model.
- CDA for document ingestion or document exchange only.

## Guidance

Treat the near-term standards as the likely language for funding,
patient-administration, and API integration work. Treat the watch-list items
as conditional until a future track defines connector scope, privacy controls,
and a clear implementation boundary.
