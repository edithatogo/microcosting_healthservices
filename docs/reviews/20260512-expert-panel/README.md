## Purpose

The panel reviews whether the project is credible as a validated, extensible,
polyglot health-funding calculation library. The reviews focus on:

- contract explicitness;
- versioned schemas and evidence;
- SAS and Excel parity;
- Rust-core and ABI strategy;
- Arrow-based data boundaries;
- security and supply-chain risk;
- costing-study support;
- policy and public-use implications;
- prioritised implementation order.
