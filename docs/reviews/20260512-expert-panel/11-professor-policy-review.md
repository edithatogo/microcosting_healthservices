# Simulated Professor of Policy Review

## Role

Reviewer focused on governance, public-sector accountability, policy language,
legitimacy, disclaimers, and appropriate use.

## Findings

1. The library must not appear to speak for IHACPA.
   It should cite IHACPA sources but remain clear that it is an independent
   implementation unless officially endorsed.

2. Policy interpretation should be conservative.
   NWAU and NEP calculations are not the whole of funding policy.

3. Documentation should state appropriate and inappropriate uses.
   Users should know when the package is suitable for research, validation,
   costing studies, or operational funding estimates.

4. Public roadmap claims should distinguish aspiration from delivered capability.
   This matters for credibility and institutional trust.

5. Licensing and data governance must be prominent.
   Classification products, groupers, and local costing data may be restricted.

## Recommended Policy Controls

- Add an "appropriate use" page to docs.
- Add disclaimers around official status and local funding interpretation.
- Add source and licensing notices near classification and costing materials.
- Add policy-sensitive release notes when a pricing-year update changes outputs.
- Add governance audit reports before public milestone claims.

## Priority Recommendation

Before broad promotion, strengthen public-facing disclaimers, validation status
language, and appropriate-use documentation.
